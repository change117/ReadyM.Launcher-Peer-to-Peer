# Uploads Directory

This directory is used to store files uploaded through the launcher.

## Usage

Files uploaded through the web interface will be stored here. In P2P mode, these files can be shared directly with connected peers.

## File Types

You can upload:
- Game files
- Mods and modifications
- Configuration files
- Save files
- Any other game-related content

## Note

This directory is included in `.gitignore` to prevent accidentally committing uploaded files to the repository.
